Rhino → WebXR

Exportiere dein Rhino-Modell als .glb oder .gltf und lade es hier. Auf einem Smartphone erscheint automatisch die AR-Schaltfläche.

Warte auf Modell…

Tipp: In Rhino das Modell über Exportieren als .glb speichern (bettet Texturen ein). Im AR-Modus das Handy langsam bewegen, bis der Ring auf einer Fläche erscheint – dann tippen, um das Modell zu platzieren.
